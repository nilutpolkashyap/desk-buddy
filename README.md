# DESK BUDDY: A Multi-Function IoT Desk Companion with Games

 Build a feature-packed desk gadget using an ESP8266, a tiny OLED screen, a motion sensor, touch buttons, and a buzzer. It tells time, shows weather, runs a Pomodoro timer, displays animated robot eyes, and packs 5 retro games — all in one small device you can interact with by tilting, tapping, and flipping.
